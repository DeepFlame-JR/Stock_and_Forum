# 주식과 종목토로방
주식 가격과 네이버 종목토론방의 내용을 일자 별로 수집

## Purpose
네이버 종목토론방의 여론과 주식 가격의 연관성 분석   
(들은 바로는 종목토론방에 부정적인 여론이 있으면 주가는 오히려 오른다고 한다.)

---
## Solution
1. 네이버 증권에서 주식 가격을 가져와 PostgreSQL에 저장한다.
2. 네이버 종목토론방의 정보를 가져와 MongoDB에 저장한다.
